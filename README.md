# MoodLoom

> Track your mood, weave your wellness.

MoodLoom is an offline-first Flutter mood-tracking and wellness journal. Log how you feel, write journal entries, and watch patterns emerge through rich insights тАФ with optional cloud sync via Supabase. It goes well beyond a simple tracker, adding AI-assisted sentiment analysis, face-based mood detection, dream journaling, breathing exercises, mood prediction, time capsules, achievements, and an app lock for privacy.

## тЬи Features

- **Mood Logging** тАФ quick 1тАУ5 mood entries with emoji, optional notes, journal text, tags, and location.
- **Journal** тАФ long-form journal entries attached to moods, with full-text search and detail views.
- **Calendar** тАФ browse and revisit your mood history day by day.
- **Insights** тАФ charts and analytics (`fl_chart`) surfacing trends, averages, and streaks.
- **Mood Map** тАФ see where your moods happen via geolocation and reverse geocoding.
- **Mood Prediction & Patterns** тАФ predicts likely mood and detects recurring patterns from your history.
- **Sentiment Analysis** тАФ analyzes journal text to infer emotional tone.
- **Face Mood Detection** тАФ estimates mood from a camera selfie using ML Kit face detection.
- **Speech-to-Text** тАФ dictate journal entries and notes by voice.
- **Dream Journal** тАФ log dreams with sleep quality, recall, and tags.
- **Time Capsules** тАФ write a message to your future self that unlocks on a chosen date.
- **Breathing Exercises** тАФ guided breathing for calming and focus.
- **Achievements** тАФ unlock badges for entry counts, streaks, and journaling milestones.
- **Mood Avatar / Ripple / Tree** тАФ playful, generative visualizations of your emotional state.
- **Custom Moods & Tags** тАФ define your own moods and organize entries with tags.
- **Discover** тАФ community/inspiration tab with sign-in gating.
- **Cloud Sync** тАФ optional Supabase auth + sync; the app is fully usable offline as a guest.
- **App Lock** тАФ PIN protection on launch for private journals.
- **Dark Mode & Onboarding** тАФ themeable UI with a first-launch onboarding flow.
- **Export & Share** тАФ export entries to CSV and share mood cards.

## ЁЯЪА Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (Dart `^3.10.7`)
- A configured Android/iOS toolchain (Android Studio / Xcode)
- A [Supabase](https://supabase.com) project (optional тАФ only needed for cloud sync & auth)

### Setup

1. **Clone & install dependencies**
   ```bash
   git clone <repo-url>
   cd moodloom
   flutter pub get
   ```

2. **Configure Supabase** (optional)

   MoodLoom works fully offline as a guest. To enable cloud sync and accounts, set up
   a Supabase project and provide its URL and anon key to `SupabaseService.initialize()`
   in [lib/services/supabase_service.dart](lib/services/supabase_service.dart) (e.g. via
   constants or `--dart-define`).

3. **Run**
   ```bash
   flutter run
   ```

### Permissions

Depending on the features you use, the app requests:

- **Camera** тАФ face-based mood detection
- **Microphone** тАФ speech-to-text journaling
- **Location** тАФ mood map / location-tagged entries

### App Icons

Launcher icons are generated by `flutter_launcher_icons`:
```bash
flutter pub run flutter_launcher_icons
```

## ЁЯз▒ Architecture

MoodLoom follows an offline-first, provider-based architecture:

- **Models** (`models/`) тАФ typed entities (`MoodEntry`, `DreamEntry`, `TimeCapsule`, `Achievement`).
- **Providers** (`providers/`) тАФ `ChangeNotifier` state for moods, settings, and tags.
- **Services** (`services/`) тАФ the heavy lifting: a local SQLite store (`DatabaseService`),
  Supabase auth/sync (`SupabaseService`, `SyncService`), and feature services for sentiment,
  face mood, prediction, patterns, speech, location, breathing, export, and app lock.
- **Screens / Widgets** (`screens/`, `widgets/`) тАФ UI that observes providers.

Data is written locally first and synced to Supabase when signed in and online тАФ so the app
stays fully functional offline, and guests can use it without an account.
